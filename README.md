# Portfolio-Data-Science-AI
Ce repository compile quelques projets et leurs rapports que j'ai pu mener individuellement et en équipe au cours de mes études. Les raports sont rédigés en anglais.

## Sujets couverts : 

### Green AI : Prédiction de potabilité de l'eau grâce aux algorithmes de machine learning et évaluation des modèles en prenant en compte les enjeux d'environnement et de santé
Le dossier *water-potability-prediction_green-ai* contient : Fichier Jupyter Notebooks, le dataset utilisé et rapport de présentation PDF.

### Étude comparative ML : Comparaison de modèles d'apprentissage d'ensemble vs apprentissage évolutionnaire sur le dataset Higgs Boson, afin de classifier la nature d'un signal (évènement ou bruit)
Le dossier *comparative-study-ensemble-evolutionary-learning* contient le fichier Jupyter Notebooks présentant le code et faisant également office de rapport. Le dataset utilisé n'a pas été uploadé à cause de son volume.

### Graph mining : Analyse de la circulation touristique avec Neo4j - Monopartite / Bi-partite - langage Cypher
Le dossier *tourism-circulation-analysis-neo4j* contient le rapport d'analyse sous forme de fichier PDF. Les datasets n'ont pas été uploadées à cause de leur volume.

### NLP : Challenge de recommandation d'hôtels basés sur des avis TripAdvisor, visant à battre BM25 et évaluation par MSE
Le dossier *hotel-recommendation-challenge-nlp* contient : Fichier Jupyter Notebooks présentant le code et le rapport PDF.

### Maintenance prédictive pour l'industrie : Utilisation de divers méthodes de machine learning afin de concevoir une solution de maintenance intelligente pour les équipements industriels à partir du dataset SECOM
Le dossier *predictive-maintenance-industrial-equipment* contient le rapport descriptif du projet sous forme de fichier PDF.
